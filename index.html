Frequently Asked Questions

Some of the most asked question in Ruby Support Server

  • Is using Ruby Legal / Fair Play?

    Yes, we always support Fair Play. Ruby collects all the data from Supercell API and calculations are based on that.

  • How cwl hitrates are calculated in ruby?

    Attack hitrate = number of 3*/total no. Of attacks
    Defense hitrate = number of def/total attacks on that base
    Townhall levels don't matter

  • How to setup default clan in a channel?

    +setup #channel #clantag 1 clan per channel, +unlink in that channel to unclaim it

  • Why Ruby is so slow?

    Image processing is a cpu heavy task and takes more time to generate than text messages and after that there is upload time. This makes the whole process slower. Still its speed is satisfactory

  • How are season points measured/assigned?

    Season points are given based on one players performance in cwl.
    It gives points based on 3 stars, 2 stars defences.
    Points gets deducted when they miss, dips (dips points are deducted from total 3/2 points). Also their are some bonus points based on cumulative matches played vs stars gained ratio.
